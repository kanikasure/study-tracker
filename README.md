# 📚 Study Tracker

A web-based productivity app to track study sessions, visualize daily progress, and monitor weekly study habits.

## 🔗 Live Demo
[Click here to view live app](https://kanikasure.github.io/study-tracker)

## ✨ Features
- Add and manage study subjects
- Start/stop study timer with live display
- Automatic session logging
- Daily pie chart showing time distribution per subject
- Weekly stacked bar chart showing study patterns
- Data persists across sessions using localStorage

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript
- Chart.js
- localStorage API

## 💡 What I Learned
